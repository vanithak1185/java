Configure Maven to execute the Clean lifecyle phases in sequential order.

Configure each of the Clean lifecycle phases to print the respective message during each of its phase execution 

 

  1. $mvn pre-clean -> Now executing the PRE-CLEAN phase of Clean Lifecycle for Artifact ID : <project artifact Id>
  2. $mvn clean -> Now executing the CLEAN phase of Clean Lifecycle for Artifact ID : <project artifact Id>
  3. $mvn post-clean -> Now executing the POST-CLEAN phase of Clean Lifecycle for Artifact ID : <project artifact Id>

Note: <project artifact Id> Read the value of artifact Id from pom.xml and display it 

 

General Instructions

 1. Select 'Run -> Install' to install Maven

 2. Select 'Run -> Run' to execute maven post-clean phase

 2. Select 'Run -> Test' to execute test cases and view the test results
